1. 简介 [show]

代码生成器为模型生成了数据代码接口。代码接口报告汇总模型可调用入口函数和数据接口元素(例如根级输入端口和输出端口)的接口代码。

2. 函数接口 [hide]

外部代码通过调用为模型生成的入口函数来启动生成代码的执行。代码生成器为模型生成的函数代码类型取决于在模型中表示的建模样式和周期性速率。有关详细信息，请参阅为模型入口函数配置生成的 C 函数接口

2.1 初始化函数

初始化入口函数实现启动行为。在模型中，Initialize Function 模块显式表示启动行为。

PendulumQuickStart_RL_initialize

生成代码的初始化入口函数
#include "PendulumQuickStart_RL.h" void PendulumQuickStart_RL_initialize(void)

2.2 终止函数

终止入口函数实现关闭行为。在模型中，Terminate Function 模块显式表示关闭行为。

PendulumQuickStart_RL_terminate

生成代码的终止入口函数
#include "PendulumQuickStart_RL.h" void PendulumQuickStart_RL_terminate(void)

2.3 周期函数

周期性入口函数实现以固定采样率发生的模型行为。对于基于速率的模型，代码生成器为模型中使用的每个速率生成一个周期性入口函数。在函数调用(导出函数)模型中，指定采样率的 Function Call Subsystem 模块表示周期函数。

PendulumQuickStart_RL_step

生成代码的输出入口函数. 必须周期性调用，每 0.004 秒调用一次.
#include "PendulumQuickStart_RL.h" void PendulumQuickStart_RL_step(void)

2.4 非周期函数

非周期(异步)入口函数实现异步发生的组件行为。在函数调用(导出函数)模型中，不指定采样率的 Function Call Subsystem 模块表示非周期函数。

模型中没有表示非周期函数。

3. 数据接口 [hide]

数据代码接口将模型数据接口元素(如根级输入端口和输出端口)映射到存储类。存储类指定生成代码中数据元素的外观和位置。存储类支持直接访问数据通信。可用于映射的存储类列表由与模型相关联的 Embedded Coder 字典确定。您可以在模型代码映射中将模型接口元素映射到存储类。

3.1 输入端口

输入是基于模型根级输入端口的存储类设定在代码中生成的变量。

模型中不存在输入端口。

3.2 输出端口

输出是基于模型根级输出端口的存储类设定在代码中生成的变量。

模型中不存在输出端口。

3.3 参数

参数是生成的代码中的变量，表示模型参数和模型参数参量。当模型代码在目标环境中执行时，将基于存储类设定生成变量并该变量标记为可调或不可调。

3.4 内部数据

内部数据是生成的代码中的变量，表示信号数据、具有内部状态的模块和数据存储。当模型代码在目标环境中执行时，将基于存储类设定生成变量并该变量标记为测量或非测量。

模型中没有全局数据存储或具有非自动存储类的数据存储。
